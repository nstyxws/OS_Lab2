### **Словник базових англійських термінів**

* **Command Line Interface (CLI)** *(Інтерфейс командного рядка)* — система для введення текстових команд та скриптів для управління комп'ютером.  
* **Command** *(Команда)* — окреме слово або інструкція, що вводиться в CLI для виконання певної дії.  
* **Script** *(Скрипт / сценарій)* — набір команд, збережених у файлі, які система аналізує (parsing) та виконує як єдине ціле.  
* **GUI (Graphical User Interface)** *(Графічний інтерфейс користувача)* — візуальне середовище операційної системи, яке використовує вікна, іконки та меню замість тексту.   
* **Terminal / Console** *(Термінал / Консоль)* — програмне або віртуальне середовище для безпосередньої взаємодії з командним рядком.  
* **Kernel** *(Ядро операційної системи)* — основний компонент ОС, що розподіляє системні ресурси (пам'ять, процесор, диск) та керує всіма додатками.  
* **Process** *(Процес)* — окреме завдання або запущена програма (чи її частина), яка відстежується та керується ядром.  
* **Multitasking** *(Багатозадачність)* — процес швидкого перемикання ядра між різними процесами, що створює ілюзію їх одночасної роботи.  
* **API (Application Programming Interface)** *(Інтерфейс програмування додатків)* — набір правил, за якими додатки звертаються до ядра за ресурсами, не переймаючись технічними деталями обладнання.

 ### ** Визначення **
 **Інтерфейс командного рядка (CLI)** \- це програмний механізм, який використовується для взаємодії з операційною системою за допомогою клавіатури. 

**Термінал на основі графічного інтерфейсу (GUI)** — це програма-емулятор, яка поєднує зручність вікон, кнопок та меню (як у вікні  
Windows) із потужністю командного рядка (CLI).

**Віртуальний термінал у Linux** \- це спосіб взаємодії з операційною системою через текстовий інтерфейс. Це як вікно, яке відображає текстовий вихід команд і приймає введення від користувача.
